# $username
Returns the username of the author of the message.

## Syntax
```
$username
```

## Example
```
$nomention
Hello $username!
```

``` discord yaml
- user_id: 696368083517964288
  username: Spen
  color: "#b9a0e0"
  content: |
    !echo

- username: Wiki Bot
  color: "#b9a0e0"
  bot: true
  content: |
    Hello Spen!
```
